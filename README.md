# Budget-Traker
App di gestione entrate e uscite sviluppata in JavaScript puro (senza framework). Permette di aggiungere, modificare ed eliminare transazioni, salvare i dati in localStorage e visualizzare il bilancio tramite grafico interattivo.

# Expense Tracker App

Applicazione web per la gestione di entrate e uscite, sviluppata in **JavaScript Vanilla** (senza framework).
---

## Funzionalità

*  Aggiunta di entrate e uscite
* Modifica delle transazioni (double click)
* Eliminazione delle voci
* Salvataggio automatico con `localStorage`
*  Grafico dinamico con Chart.js
* Calcolo saldo in tempo reale
* Data e ora per ogni operazione
* Interfaccia moderna stile app bancaria

---

## 🛠 Tecnologie utilizzate

* HTML5
* CSS3
* JavaScript (Vanilla)
* Chart.js

---

## Grafico

Il grafico mostra la distribuzione tra entrate e uscite, aggiornandosi automaticamente ad ogni modifica.

---

##  Come funziona

1. Inserisci descrizione e importo
2. Seleziona il tipo (entrata/uscita)
3. Premi "Aggiungi"
4. I dati vengono salvati automaticamente

---

##  Salvataggio dati

I dati vengono salvati nel browser tramite:

* `localStorage`
